# assignment2-template

Create your own repository from the assignment2-template repo. Name your repository as “assignment2-your-iu-username”. Make sure the repository is “Private”.   

Once your own repository is created from the template, clone your repository and use the provided starter code for each problem to implement the requirements outlined below. 

After implementing your coding solutions push your changes to the newly created repository and submit the commit URL to the corresponding assignment page on Canvas.
